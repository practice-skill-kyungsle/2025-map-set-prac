## 📝 문제 1 (난이도: ⭐)

다음 배열에서 **중복된 숫자를 제거**하고, 고유한 숫자들만 남긴 새로운 배열을 만들어주세요.

```javascript
const numbers = [1, 2, 3, 2, 4, 1, 5, 3, 6, 4];

// 여기에 코드 작성
// 결과: [1, 2, 3, 4, 5, 6]
```

**힌트**: 어떤 자료구조가 "중복 제거"에 특화되어 있을까요?

답변을 작성해주세요! 😊


## 답변 

```js
const numbers = [1, 2, 3, 2, 4, 1, 5, 3, 6, 4];
const newSet = new Set(numbers)
const result = Array.from(newSet)
console.log(result)
```
이건가?

## 조언 

```js
const numbers = [1, 2, 3, 2, 4, 1, 5, 3, 6, 4];
const result = [...new Set(numbers)];
console.log(result);
```
