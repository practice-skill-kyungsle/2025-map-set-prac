## 📝 문제 6 (난이도: ⭐⭐⭐⭐)

전화번호부 시스템을 만들어주세요. 한 사람이 여러 개의 전화번호를 가질 수 있고, 전화번호로 역검색도 가능해야 합니다.

```javascript
// 구현해야 할 기능:
// 1. addContact(name, phoneNumber) - 연락처 추가
// 2. getPhoneNumbers(name) - 이름으로 전화번호들 조회
// 3. findName(phoneNumber) - 전화번호로 이름 찾기

// 테스트:
addContact("철수", "010-1111-1111");
addContact("철수", "010-2222-2222");
addContact("영희", "010-3333-3333");

getPhoneNumbers("철수"); // ['010-1111-1111', '010-2222-2222']
findName("010-3333-3333"); // '영희'
```

## 답변

```js
const phoneNumMap = new Map();
const addContact = (name, phoneNum) => {
  if (!phoneNumMap.get(name)) phoneNumMap.set(name, []);
  phoneNumMap.get(name).push(phoneNum);
};
const getPhoneNumbers = (name) => console.log(phoneNumMap.get(name));
const findName = (phoneNum) => {
  for (let entry of phoneNumMap) {
    if (entry[1].includes(phoneNum)) {
      console.log(entry[0]);
      break;
    }
  }
};
```

## 조언

```js
const phoneNumMap = new Map();
const reverseMap = new Map(); // 전화번호 -> 이름 매핑

const addContact = (name, phoneNum) => {
  if (!phoneNumMap.has(name)) phoneNumMap.set(name, []);
  phoneNumMap.get(name).push(phoneNum);
  reverseMap.set(phoneNum, name); // 역검색용
};

const findName = (phoneNum) => reverseMap.get(phoneNum);
```
