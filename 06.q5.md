## 📝 문제 5 (난이도: ⭐⭐⭐)

문자열에서 각 문자가 몇 번 등장하는지 세는 함수를 만들어주세요.

```javascript
const str = "hello world";

// 여기에 코드 작성

// 결과 예시:
// h: 1
// e: 1
// l: 3
// o: 2
// (공백과 나머지 문자들도...)
```

## 답변

```js
const str = "hello world";

const wordCounter = (str) => {
  const wordMap = new Map();
  for (let word of str) {
    if (!wordMap.get(word)) {
      wordMap.set(word, 1);
    } else {
      wordMap.set(word, wordMap.get(word) + 1);
    }
  }
  wordMap.forEach((value, key) => {
    console.log(`${key}: ${value}`);
  });
};

wordCounter(str);
```
